# Demo data — 100% fictional

Everything in this folder is **invented sample data** for the fictional
**Sample Company LLC**. The company, the client names (Maple Street Dental,
Harbor Fitness, Bluebird Bakery, Cedar Point Vet Clinic, Riverside Cafe), and
every amount are made up for teaching. Any resemblance to real businesses is
coincidental.

| File | What it is |
| --- | --- |
| `accounts.csv` | 22-account chart of accounts |
| `trial_balance_2025-12-31.csv` | Prior-year ending trial balance (post-close, balances at $38,700.00) |
| `transactions_q1_2026.csv` | 47 transactions, January–March 2026 |

Import them in that order via the app's **Get Started** page — or click
**⚡ Load all the demo data for me**.

**Never put real client data in this repo** — not in these files, not in new
ones, not in screenshots. If you want a different dataset, invent one (see the
"second demo company" issue).
